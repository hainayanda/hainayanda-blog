# hainayanda-blog
My Personal Blog
Production: https://hainayanda.com/

## angular
### installation
- navigate to hainayanda-blog
- npm install
### build
- navigate to hainayanda-blog
- ng build --prod

## server
### installation
- navigate to hainayanda-blog/server
- npm install

### build
- navigate to hainayanda-blog/server
- npm run tsc

## production run
- navigate to hainayanda-blog
- npm start
